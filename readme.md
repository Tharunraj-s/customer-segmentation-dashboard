# Customer Segmentation & Sales Analytics Dashboard

An interactive R Shiny dashboard for analyzing customer purchasing behavior and performing customer segmentation.

## Dataset
Online Retail Dataset

## Features
- Sales trend analysis
- Country-level sales performance
- Top products and customers
- Customer demographic insights
- Customer segmentation using K-means clustering
- RFM analysis

## Tech Stack
- R
- Shiny
- dplyr
- ggplot2
- lubridate
- cluster

## Pipeline
1. Data cleaning
2. Feature engineering
3. Customer summary creation
4. Interactive Shiny dashboard
